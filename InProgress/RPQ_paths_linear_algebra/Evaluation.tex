Hardware

\begin{itemize}

	\item \textbf{RQ2:} How does our implementation compare with production graph database systems?
	\item \textbf{RQ3:} What is the performance behavior across different query types and graph structures?
\end{itemize}

\subsection{Experimental Results}

Experimental results will be presented here. Based on preliminary findings from our earlier work \cite{!!!}, we expect the following:

\begin{itemize}
	\item The proposed algorithm should demonstrate significant speedups over existing linear algebra-based solutions (RPQ-matrix) on complex queries involving Kleene star operators;
	\item The algorithm should achieve comparable or better performance than MillenniumDB on reachability queries;
	\item Performance should be competitive with FalkorDB while supporting a broader class of regular path queries.
\end{itemize}
