\section{Introduction}

Intro. FLPQ~\cite{doi:10.1137/S0097539798337716}

Importance of 2-RPQs in graph databases (GQL, Cypher, SPARQL).
Performance bottleneck and the promise of sparse linear algebra. 
The original LARPQ algorithm.

The original LARPQ algorithm solves the reachability problem (single-source).
Many real-world applications require more: counting paths, finding the shortest path, or ranking paths.
Existing linear algebra-based RPQ solutions (like LARPQ and RPQ-matrix) are not designed for these tasks.

Core idea: generalize the Boolean algebra used in LARPQ to an arbitrary semiring.
This transforms the algorithm from a simple BFS for reachability into a framework for computing a wide range of path-aggregation problems.

Contributions:
\begin{itemize}
    \item A formal generalization of the LARPQ algorithm to a semiring-based framework (Semiring-LARPQ).
    \item Complexity and proof of correctness for the generalized algorithm.
    \item  Instantiation of the framework with specific semirings to solve distinct problems: !!!.
    \item An experimental evaluation comparing Semiring-LARPQ against the original LARPQ and other systems (MillenniumDB) for these new problem classes, demonstrating its efficacy and performance.
\end{itemize}
